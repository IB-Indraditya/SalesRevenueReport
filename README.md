## 🏢 Project Overview

The **Sales Revenue Tracking System** is a centralized, automated Excel-based Management Information System (MIS) designed to monitor sales performance, revenue generation, profitability, and business trends in real time.

The workbook is built to serve as a **single source of truth**, where every sales transaction entered into the system automatically updates dashboards, KPI cards, charts, and analytical reports without requiring any manual calculations.

This solution enables business leaders to make faster, data-driven decisions through dynamic reporting and interactive analytics.

# ➕ Adding a New Sales Transaction

Follow these steps to record a new sale:

### Step 1

Navigate to the **Sales_Data** worksheet.

### Step 2

Scroll to the first available blank row.

### Step 3

Enter values only in the editable columns:

* Sale ID
* Transaction Date
* Region
* Sales Representative
* Product Category
* Product Name
* Units Sold
* Unit Price
* Unit Cost

### Step 4

The following fields are calculated automatically:

* Revenue
* Total Cost
* Gross Profit
* Profit Margin

**Do not overwrite these calculated columns.**

### Step 5

Once the record is saved, all dashboards, charts, KPIs, Pivot Tables, and summary reports will refresh automatically.

# 📌 Best Practices

To ensure accurate reporting:

* Always enter data in the **Sales_Data** worksheet.
* Do not delete or rename worksheets.
* Do not overwrite formula cells.
* Update only the **Target** column in the **Targets_vs_Actual** sheet.
* Save the workbook after every data update.
* Refresh Pivot Tables if manual refresh is enabled.

---

# 🔐 Data Integrity Guidelines

To maintain workbook integrity:

* Avoid modifying formulas.
* Avoid changing worksheet names.
* Avoid inserting columns within system-generated tables.
* Use consistent date formats (DD-MM-YYYY).
* Ensure Sale IDs remain unique.
