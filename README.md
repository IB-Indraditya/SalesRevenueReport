# SalesRevenueReport

  Sales Revenue Tracker System — User Guide				
				
				
	Purpose			
	This workbook is a complete, self-updating Sales Revenue Tracking system. Log every sale in one place and every report, KPI and chart in the workbook recalculates automatically.			
				
	Who is this for			
	Sales Operations, Regional Managers, and the Sales Manager reviewing team performance.			
				
				
	Workbook Map			
	Sheet	What it contains	Do you edit it?	
	Dashboard	Executive summary: KPIs, revenue trend, region/category/rep breakdowns.	No — auto-updates	
	Sales_Data	One row per transaction — the single source of truth. Add new sales here.	Yes — add new rows here	
	Monthly_Summary	Revenue, cost, profit and margin rolled up by month.	No — auto-updates	
	Regional_Summary	Performance broken down by sales region.	No — auto-updates	
	Category_Summary	Performance broken down by product category.	No — auto-updates	
	Rep_Performance	Individual sales rep scorecard and ranking.	No — auto-updates	
	Targets_vs_Actual	Monthly revenue target vs. actual achievement.	Yes — edit Target column only	
				
				
	How to add a new sale			
	1. Open the Sales_Data sheet.			
	2. Go to the first empty row at the bottom of the table.			
	3. Fill in Sale ID, Date, Region, Sales Rep, Category, Product, Units Sold, Unit Price and Unit Cost (blue cells only).			
	4. Revenue, Cost and Profit calculate automatically — do not type into those columns.			
	5. The Dashboard, Monthly_Summary, Regional_Summary, Category_Summary and Rep_Performance sheets update instantly.			
				
	Color Key			
	Blue text	Editable input — safe to type over		
	Black text	Formula — do not overwrite		
	Green text	Link pulling from another sheet
